# Simple Dark Mode Calculator 🖤✨

A sleek web-based calculator with **Light/Dark mode**. Designed to be **compact, responsive, and user-friendly**, perfect for quick calculations on any device.  

---

## Features

- **Basic arithmetic:** addition (+), subtraction (-), multiplication (×), division (÷)  
- **Special functions:**  
  - `C` (clear)  
  - `=` (equals)  
  - `+/-` (toggle sign)  
  - `√` (square root)  
  - `%` (percentage)  
- **Light/Dark mode toggle** for easy theme switching  
- **Compact and responsive design** for all screen sizes  
- **Smooth hover animations** on all buttons

## Usage

1. Open `index.html` in any modern browser.  
2. Click the buttons to perform calculations.  
3. Toggle **Dark Mode** using the button in the header.
